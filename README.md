## Madhav Ecommerce Sales Dashboard (Power BI)

## Project Overview
This project analyzes Madhav Ecommerce sales data using Power BI. The dashboard provides interactive visualizations to understand sales performance, profit trends, and customer purchasing behavior. It helps businesses gain valuable insights and make data-driven decisions.

## Tools Used
* Power BI
* Excel / CSV Dataset
* Data Visualization

## Key Features
* Interactive dashboard with filters and slicers
* KPI cards for Sales, Profit, Quantity, and AOV
* Sales analysis by State
* Monthly Profit and Loss analysis
* Category and Sub-Category performance
* Payment Mode analysis
* Top Customers analysis

## Key Insights
* Clothing category has the highest quantity share.
* COD (Cash on Delivery) is the most used payment mode.
* Maharashtra and Madhya Pradesh generate the highest sales.
* Printers and Bookcases are the most profitable sub-categories.

## Conclusion
This dashboard helps analyze sales performance, identify profitable products, and understand customer purchasing behavior to support better business decisions.
